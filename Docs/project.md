# SUSHI CHOICE Brand Platform v2.0

## Project Vision

대한민국 회전초밥 업계에서 가장 세련되고 고급스러운 브랜드 플랫폼 구축

단순 음식점 홈페이지가 아닌

* 브랜드 경험 플랫폼
* 회전초밥 전문 플랫폼
* 가맹 모집 플랫폼
* 채용 플랫폼
* 지역 SEO 플랫폼
* AI 검색(GEO) 최적화 플랫폼

으로 구축한다.
