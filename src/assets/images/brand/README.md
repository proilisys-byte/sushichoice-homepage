# 🏷️ 브랜드 이미지

## 필요한 파일

### 로고
- `logo-white.svg` 또는 `logo-white.png` — 흰색 로고 (다크 배경용)
- `logo-gold.svg` 또는 `logo-gold.png` — 골드 로고 (선택)
- `logo-dark.svg` 또는 `logo-dark.png` — 다크 로고 (밝은 배경용)

### 기타
- `og-image.jpg` — SNS 공유 시 표시될 대표 이미지 (1200×630 권장)

## 이미지 가이드
- 로고: SVG 형식 우선, PNG 투명 배경도 OK
- OG 이미지: 브랜드 로고 + 대표 메뉴 조합
