# 👨‍🍳 Chef's Craft 섹션 이미지 (장인의 온도 ⭐)

## 필요한 파일 (3장 — 3단 스토리별 1장)

### Part 1 — 샤리의 온도 (36.5°C)
- `chef-shari.jpg` — 셰프의 손이 샤리(초밥밥)를 쥐는 매크로 클로즈업
  - 손끝의 디테일이 보이면 좋음
  - 따뜻한 조명 톤

### Part 2 — 칼의 각도 (一刀一點)
- `chef-knife.jpg` — 셰프의 칼로 생선을 손질하는 순간
  - 칼날 + 생선 결이 보이는 클로즈업
  - 날카롭고 정밀한 느낌

### Part 3 — 쌀의 집착
- `chef-rice.jpg` — 가마솥 밥 또는 쌀알 클로즈업
  - 쌀, 소금, 식초 등 재료가 보이면 좋음
  - 깨끗하고 정성스러운 느낌

## 이미지 가이드
- 해상도: 1200×1600 이상 (세로형 또는 정방형도 OK)
- 다크 우드 톤 배경이면 가장 잘 어울림
- 매크로/클로즈업 촬영 권장
- 셰프의 손이 주인공이 되는 구도
