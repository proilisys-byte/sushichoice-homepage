# 🍣 Rotary Experience 섹션 이미지

## 필요한 파일 (4장 — 각 스텝별 1장)

### Step 1 — 보다 (SEE)
- `rotary-see.jpg` — 다양한 초밥이 레일 위를 지나가는 와이드 숏
  - 황금빛 레일 + 컬러풀한 초밥들이 보이면 좋음

### Step 2 — 고르다 (CHOOSE)
- `rotary-choose.jpg` — 손이 접시를 집어 드는 클로즈업
  - 자연스러운 손 + 접시 집는 순간

### Step 3 — 주문하다 (ORDER)
- `rotary-order.jpg` — 태블릿 터치 또는 셰프가 칼을 드는 장면
  - 주문 과정을 보여주는 이미지

### Step 4 — 즐기다 (ENJOY)
- `rotary-enjoy.jpg` — 가족/고객이 함께 웃으며 식사하는 따뜻한 장면
  - 밝고 따뜻한 톤

## 이미지 가이드
- 해상도: 1200×800 이상
- 가로형(3:2 또는 16:9) 권장
- 매장 실제 사진 권장
