# 🎬 Hero 섹션 이미지/영상

## 필요한 파일

### 배경 영상 (우선순위 1)
- `hero-video.mp4` — 시네마틱 루프 영상 (10~15초)
  - 연어 초밥 제작, 광어 손질, 토치 초밥, 회전레일 위 초밥
  - 해상도: 1920×1080 이상 (Full HD 권장)
  - 파일 크기: 10MB 이하 (웹 최적화)

### 배경 이미지 (영상 대체 또는 폴백)
- `hero-bg-1.jpg` — 레일 위 초밥이 흐르는 와이드 숏
- `hero-bg-2.jpg` — 매장 전경 (골드빛 조명)
- `hero-bg-3.jpg` — 셰프가 초밥 쥐는 모습
  - 해상도: 1920×1080 이상
  - 어두운 톤 권장 (다크 오버레이 적용됨)

## 이미지 가이드
- 가로형(16:9) 비율 권장
- 어두운 배경 또는 어두운 톤의 사진이 좋음 (밝은 텍스트가 올라감)
- 고화질, 선명한 이미지 사용
